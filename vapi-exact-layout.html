Get Started > Introduction

Welcome to VoicePartner AI

VoicePartner AI is a developer platform for building, testing, and deploying advanced voice AI agents in minutes. Create conversational AI that can handle phone calls, web interactions, and complex workflows.

ðŸš€ Key Features

  • Real-time Conversations: Build responsive voice agents
  • Phone & Web Integration: Support for both phone calls and web interactions
  • Flexible Architecture: Use Assistants for simple tasks or Squads for complex workflows
  • API Connectivity: Connect to your existing systems and databases

Getting Started

Follow these steps to create your first voice assistant:

  1. Create an assistant with a system prompt
  2. Configure voice and model settings
  3. Add a phone number or web integration
  4. Test and deploy your assistant
// Example: Creating an assistant via API
const assistant = await vapi.assistants.create({
  model: {
    provider: "openai",
    model: "gpt-4o-mini",
    systemPrompt: "You are a helpful customer service assistant."
  },
  voice: {
    provider: "elevenlabs",
    voiceId: "elliot"
  }
});
Core Features > Assistants

Assistants

Assistants are the core building blocks of VoicePartner AI. They handle conversations with users through voice interactions, powered by large language models and configurable with custom behavior.

Creating an Assistant

Every assistant needs three key components:

  • Model: The language model that powers conversations
  • Voice: Text-to-speech configuration
  • System Prompt: Instructions that define behavior

Configuration Options

Model Settings

Choose from providers like OpenAI, Anthropic, or Google. Configure temperature, max tokens, and response behavior.

Voice Configuration

Select voices from ElevenLabs, OpenAI, or other providers. Adjust speed, pitch, and speaking style.

Advanced Features

Add tools, configure transcription settings, set up analysis and monitoring.

Core Features > Tools

Tools

Tools extend your assistant's capabilities beyond conversation. They allow your assistant to interact with external systems, APIs, and perform actions based on user requests.

Types of Tools

ðŸ“ž Default Tools

Common Use Cases

  • Sending follow-up emails or SMS messages
  • Processing orders and payments
  • Managing support tickets

Get Started > CLI Quickstart

CLI Quickstart

Get started with VoicePartner AI using our command-line interface (CLI) for rapid development and testing.

Installation

# Install the VoicePartner AI CLI
npm install -g @voicepartner/cli

# Verify installation
vapi --version

Authentication

# Login with your API key
vapi auth login

# Or set API key directly
export VAPI_API_KEY="your-api-key-here"

Quick Commands

Create Assistant

vapi assistant create \
  --name "My Assistant" \
  --prompt "You are a helpful assistant"

List Assistants

vapi assistant list

Test Call

vapi call test --assistant-id abc123
Best Practices > Prompting Guide

Prompting Guide

Master the art of writing effective prompts for voice AI assistants. Good prompts are the foundation of natural, helpful conversations.

Prompt Structure Best Practices

ðŸ“ Essential Components

  1. Identity: Who is the assistant?
  2. Context: What's the situation/business?
  3. Behavior: How should they act?
  4. Constraints: What are the limits?

Prompt Examples

âŒ Poor Prompt

You are a customer service agent. Help customers.

Issues: Too vague, no specific instructions, no personality

âœ… Excellent Prompt

You are Sarah, a friendly customer service representative for TechCorp.

PERSONALITY: Warm, professional, solution-oriented
ROLE: Answer CRM questions, handle billing, schedule demos  
STYLE: Concise responses, ask clarifying questions
CONSTRAINTS: Don't promise refunds, escalate complex issues

Voice-Specific Considerations

ðŸ—£ï¸ Natural Speech

  • Include speech patterns like "um", "let me check"
  • Use contractions: "I'll" instead of "I will"
  • Add breathing room with pauses

ðŸ“ž Phone Etiquette

  • Always start with a greeting
  • Repeat important information
  • Ask for confirmation on key details
